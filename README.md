# Case Converter

> Convert text between camelCase, snake_case, kebab-case, CONSTANT_CASE, and more.

**Live demo:** https://0xelitesystem.github.io/case-converter/

Single HTML file. Runs in the browser with no build step, no server, no tracking, and no data leaving the page.

## What it does

- Splits input on spaces, hyphens, underscores, dots, and camel boundaries
- Outputs ten cases at once: camel, Pascal, snake, kebab, constant, dot, Title, Sentence, lower, upper
- Updates live as you type
- Copies any single result with one click

## What it is not

- Not a translator or transliterator. It recases the characters you give it
- Not a code formatter. It converts identifiers and short strings, not whole files

## Use it

Open the hosted page: https://0xelitesystem.github.io/case-converter/

Or download `index.html` and open it in any browser. It works offline.

## Privacy

Everything runs client-side. No analytics, no cookies, no network calls, no local storage.
